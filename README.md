# Scientific Calculator

## 📌 Project Overview

This project is a **Scientific Calculator** built using **HTML, CSS, and JavaScript**. It performs both basic and scientific mathematical operations while maintaining a calculation history.

The calculator has a modern glassmorphism UI and is responsive, making it usable on both desktop and mobile devices.

---

## ✨ Features

### Basic Operations

* Addition (+)
* Subtraction (−)
* Multiplication (×)
* Division (÷)
* Decimal numbers

### Scientific Functions

* Parentheses `()`
* Pi (π)
* Power (^)
* Factorial (!)
* Percentage (%)
* Sine (sin)
* Cosine (cos)
* Tangent (tan)
* Logarithm (log)

### Extra Features

* Clear display (C)
* Backspace (⌫)
* Calculation history
* Clear history option
* Responsive design

## ⚙️ How It Works

1. Enter numbers and operators using the calculator buttons.
2. Click **=** to evaluate the expression.
3. Every calculation is saved in the **History** section.
4. Use **C** to clear the display.
5. Use **⌫** to remove the last character.
6. Click **Clear** in the History section to remove all previous calculations.
